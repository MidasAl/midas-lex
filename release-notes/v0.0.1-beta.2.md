# Midas Lex v0.0.1-beta.2

Prerelease for GitHub binary distribution testing.

This release attaches locally private-built Midas Lex runtime binaries, public wrapper binaries, and same-name SHA-256 checksum files.

## Automatic updates

On a default invocation with a verified runtime already installed, the wrapper starts that runtime before it may launch the hourly background check for both wrapper and runtime updates. Linux and macOS verify and atomically replace an older wrapper; Windows keeps automatic runtime updates but directs wrapper updates to `cargo install midas-lex --force` after Midas Lex exits. Update failures do not change the running command.

First runs and explicit version selectors do not start the background check. The wrapper owns no update command, and unrecognized leading `+` tokens pass through to the runtime unchanged.

Supported targets:

- `x86_64-unknown-linux-musl`
- `aarch64-unknown-linux-musl`
- `x86_64-apple-darwin`
- `aarch64-apple-darwin`
- `x86_64-pc-windows-msvc`
- `aarch64-pc-windows-msvc`

Checksums are attached as separate files and are not repeated in this release body.

Wrapper crate version for this release: `0.0.1-beta.2`.

By using this CLI, users agree to the EULA available through `midas-lex eula`.
